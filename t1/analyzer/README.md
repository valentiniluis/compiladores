# Construção de Compiladores - Trabalho 1

## Analisador Léxico

O presente projeto é atividade avaliativa da disciplina de Construção de Compiladores, do semestre 2026.1 (noturno). A proposta é o desenvolvimento de um analisador léxico que, por meio de um autômato finito, interpreta tokens de uma linguagem pre-definida.

A linguagem escolhida para a realização da atividade é um subconjunto da linguagem Python: a declaração de funções e seus parâmetros.

## Execução

Para executar o projeto, você deve ter o npm (Node Package Manager) instalado no seu dispositivo.

O código é na linguagem Typescript, que é compilada para javascript e então executada usando node.js.

Para executar o analisador léxico, execute as duas linhas abaixo num terminal, no diretório deste README.md (_/compiladores/t1_):

```
npm install
npm run dev
```
